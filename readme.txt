==============installation==============
1.run anaconda prompt
2.install flask by writing "ip install flask" and click enter
3.Install flask_bootstrap by writing "pip install flask_bootstrap" and click enter

==============Run Prototype=============
1.run anaconda prompt
2.write "cd (folder path)" and click enter
3.write "python web.py" and click enter
4.copy the link ( https://......) from command and paste to browser.
